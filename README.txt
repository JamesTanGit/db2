1. Install XAMPP Server
2. Drag and drop the entire folder 'db2' into xampp/htdocs
3. Go into XAMPP phpMyAdmin and create a database called 'db2'
4. Go into the 'SQL' tab and run the SQL code located inside of db2/DB2.sql
5. Go to localhost/db2 and start the web app by clicking on 'HomePage.html'
